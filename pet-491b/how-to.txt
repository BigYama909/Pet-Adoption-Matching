SERVER

-in terminal go to server folder and install packages list below
npm i express mongoose dotenv nodemon cors
npm i jsonwebtoken joi joi-password-complexity
npm i bcrypt
npm i nodemailer 



CLIENT

-in terminal go to client folder and install packages list below
npm i axios react-router-dom



-After install all packages do the steps below
Remove the node_modules directory so that we get a fresh copy of packages
Remove the package-lock.json 
Now clear the cache, we need to run the npm cache clean --force command in terminal. 
Run npm i
